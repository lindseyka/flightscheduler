Services

A brief overview of Flight Scheduler's capabilities.

Simple

Receive notifications of changes to your individual flight schedule, streamlined process to request flight slots, and easily notify the scheduler of days you're unable to fly due to flight physicals, appointments, etc.

Fully Featured

Track flights by day, week, month, individual crewmember. Maintenance can pull summaries of scheduled flight hours, Instructor Pilots can request crews/dates for APARTs, RL Progression, etc. Access via this site, the free iPhone/Android apps, or pull an offline version from the Cloud for access anywhere.

Secure

User accounts with different permissions only enables administrators and schedulers to manipulate the active schedule. Access is password-secure to eliminate outside infiltration.

Setup

How to get started.

  • Create a Login

    Each individual account is unique, with different permissions within your unit or company.

  • Set up your Profile

    A simple profile page allows you to select your user role, change notification settings, and set user preferences.

  • Grab the App

    Always free, the Flight Scheduler app will push notifications to you in case of a schedule change, let you view the schedule anywhere, and request/approve changes away from the office or computer.
